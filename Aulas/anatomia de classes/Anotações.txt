- Todo arquivo .java deve se iniciar com Letra maiúscula. 
Se for nome composto, o segundo nome também deve iniciar em letras maiúsculas. Exemplo: MinhaClasse

- O nome do arquivo deve ser igual ao nome da Classe

- Nome de variável - toda variável deve ser em letra minúscula. Se for palavra composta, a partir da segunda, 
deve se iniciar em letra maiúscula. Exemplo: ano, anoFrabricação

 - Estrutura básica inicial:

 /* 
// Toda classe começa como public class NomeDaClasse
public class NomeDaClasse {
  // void apenas para executar
  public static void main (String [] args) {
    
    System.out.println ("Olá mundo!");
    
  }

}
 */

 - Identação não é obrigatória como em Python. Mas, é uma boa prática para deixar o código legível.
